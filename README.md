# olist-ecommerce-sales-analysis
E-Commerce Sales Analysis using Python, Pandas, Matplotlib, and Power BI Dashboard

## 📌 Project Overview

This project analyzes the Olist Brazilian E-Commerce dataset to discover sales trends, customer behavior, regional performance, and product insights. The analysis was performed using Python and visualized through a Power BI dashboard.
## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI
- Google Colab
## 📊 Key KPIs
- Total Revenue: **14.27M BRL**
- Total Orders: **99K**
- Top Category: **Health & Beauty**
- Best Month: **May**
## 📈 Visualizations
- Revenue by Product Category
- Monthly Sales Trend
- Regional Sales
- Order Value Distribution
- Customer Review Score Distribution
- Category vs Month Heatmap
- Power BI Dashboard
## 💡 Business Insights
1. Health & Beauty generated the highest revenue.
2. May recorded the highest sales.
3. São Paulo (SP) contributed the largest regional revenue.
4. Most customers gave 5-star reviews.
5. Revenue is concentrated in a few product categories.
## 📂 Dataset
Olist Brazilian E-Commerce Dataset (Kaggle)
